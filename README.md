# WPF Analog Clock Control

Кастомный элемент управления (User Control) аналоговых часов, разработанный на платформе WPF (.NET).

## 🛠 Особенности реализации и технологии
* **Динамическое масштабирование:** Интерфейс часов (циферблат, стрелки) полностью построен на векторной графике (XAML), что позволяет корректно изменять размер окна без деформации элементов.
* **Кастомизация через Dependency Properties:** Использованы свойства зависимостей для гибкой настройки цветов элементов управления и параметров часов.
* **Функционал:** Логика плавного движения стрелок, привязка к системному времени и встроенная функция будильника с уведомлением.
* **Стек:** C#, .NET, WPF, XAML.
